# برنامج التبرعات البسيط

## ايه ده؟
ده برنامج بسيط للتبرعات زي GoFundMe بس يشتغل في الكونسول

## ازاي اشغله؟
```
python simple_main.py
```

## ايه اللي يعمله؟

### تسجيل المستخدمين:
- الاسم الاول والاخير
- الايميل (لازم يكون صحيح)
- كلمة المرور مع التأكيد
- رقم الموبايل المصري (01xxxxxxxxx)

### المشاريع:
- انشاء مشروع جديد
- عرض كل المشاريع
- تعديل مشاريعي بس
- حذف مشروع
- البحث بالتاريخ

## مثال للاستخدام:
1. اختار رقم 1 عشان تسجل
2. ادخل بياناتك
3. اختار رقم 2 عشان تدخل
4. انشئ مشروعك الاول

## ملاحظات:
- البيانات تتحفظ في الذاكرة بس (مش في ملف)
- لما تقفل البرنامج كل حاجة هتضيع
- ده برنامج تعليمي بسيط

## المشاكل الشائعة:
- لو مش شغال تأكد ان Python مثبت
- لو في ايرور في الايميل تأكد من وجود @ و .
- رقم الموبايل لازم يبدأ ب 01 ويكون 11 رقم

تم عمل البرنامج ده كمشروع تعليمي بسيط
